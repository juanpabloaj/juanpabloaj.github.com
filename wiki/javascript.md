### Arrays
Regular

    var nums = new Array();
    nums[0] = "cero";
    nums[1] = "uno";

Condensado

    var nums = new Array("cero", "uno");

Literal

    var nums = ["cero", "uno"];

### Porque usar === y no ==

    > false  === ''
    false
    > false == ''
    true
