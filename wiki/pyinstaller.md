## Install

### Windows

En Windows es instalable desde pip, pero depende de la libreria pywin32

## Uso

Crear un .exe

    pyinstaller.exe --noconsole --onefile --name=script script.py

Si se cuenta con el .spec

    pyinstaller.exe script.spec
